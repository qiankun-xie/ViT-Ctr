# Plan 02-01: LHS Parameter Sampling and Parallel Dataset Generation

## Objective

实现Latin Hypercube采样器和joblib并行ODE生成管线，生成4个HDF5文件（按RAFT类型分类），每个包含~250K带噪声的ctFP样本。

## Tasks

### Task 1: 实现LHS参数采样器和噪声注入

<read_first>
- .planning/phases/02-large-scale-dataset-generation/02-CONTEXT.md (决策D-01至D-06)
- .planning/phases/02-large-scale-dataset-generation/02-RESEARCH.md (LHS实现要点)
- src/raft_ode.py (Phase 1输出，ODE模拟器接口)
- src/ctfp_encoder.py (Phase 1输出，编码器接口)
</read_first>

<action>
创建 `src/dataset_generator.py`，包含以下函数：

1. **`generate_lhs_parameters(n_samples, raft_type)`**
   - 使用 `scipy.stats.qmc.LatinHypercube(d=7)` 生成7维连续参数采样
   - 参数范围（对数均匀采样）：
     - `log10_Ctr`: -2 到 4 (Ctr = 0.01 到 10000)
     - `log10_kp`: 2 到 4 (kp = 100 到 10000 L/mol·s)
     - `log10_kt`: 6 到 9 (kt = 1e6 到 1e9 L/mol·s)
     - `log10_kd`: -6 到 -4 (kd = 1e-6 到 1e-4 s⁻¹)
     - `I0`: 0.001 到 0.05 M (线性采样)
     - `f`: 0.5 到 0.8 (线性采样)
     - `log10_cta_ratio`: -3 到 -1 ([CTA]₀/[M]₀ = 0.001 到 0.1)
   - 返回参数字典列表，每个包含上述7个参数 + `raft_type`

2. **`inject_noise(Mn_array, D_array, sigma=0.03)`**
   - 对Mn和Đ数组注入乘性高斯噪声
   - 实现: `Mn_noisy = Mn * (1 + np.random.normal(0, sigma, size=Mn.shape))`
   - 同样处理Đ数组
   - 确保Đ >= 1.0（clip负值）
   - 返回加噪后的(Mn_noisy, D_noisy)

3. **`simulate_single_sample(params)`**
   - 调用 `raft_ode.simulate()` 生成Mn和Đ曲线
   - 调用 `inject_noise()` 加噪声
   - 调用 `ctfp_encoder.encode()` 生成64×64×2指纹
   - 返回: `{'fingerprint': fp, 'labels': [log10_Ctr, inhibition, retardation], 'params': params, 'success': True/False}`
   - 失败时返回 `{'success': False, 'params': params, 'error': str(e)}`
</action>

### Task 2: 实现并行生成和HDF5存储

<read_first>
- src/dataset_generator.py (Task 1输出)
- .planning/phases/02-large-scale-dataset-generation/02-CONTEXT.md (决策D-07至D-10)
- .planning/phases/02-large-scale-dataset-generation/02-RESEARCH.md (HDF5存储设计)
</read_first>

<action>
在 `src/dataset_generator.py` 中添加：

1. **`generate_dataset_parallel(raft_type, n_samples=250000, output_path='data/')`**
   - 调用 `generate_lhs_parameters(n_samples, raft_type)` 获取参数列表
   - 使用 `joblib.Parallel(n_jobs=-1, prefer='processes')` 并行调用 `simulate_single_sample()`
   - 集成 `tqdm` 显示进度: `from tqdm import tqdm` 和 `tqdm(total=n_samples)`
   - 收集成功样本和失败样本
   - 计算失败率: `failure_rate = len(failed) / n_samples`
   - 如果 `failure_rate > 0.05`: 打印警告并记录失败参数到 `{output_path}/{raft_type}_failures.log`
   - 调用 `save_to_hdf5()` 保存成功样本

2. **`save_to_hdf5(samples, raft_type, output_path='data/')`**
   - 创建HDF5文件: `{output_path}/{raft_type}.h5`
   - 使用 `h5py.File()` 创建3个数据集:
     - `fingerprints`: shape=(n_success, 64, 64, 2), dtype=float32, chunks=(1000, 64, 64, 2)
     - `labels`: shape=(n_success, 3), dtype=float32  # [log10_Ctr, inhibition, retardation]
     - `params`: 结构化数组存储所有输入参数（用于调试）
   - 批量写入数据（每批1000样本）避免内存溢出

3. **`main()` 函数**
   - 遍历4种RAFT类型: `['dithioester', 'trithiocarbonate', 'xanthate', 'dithiocarbamate']`
   - 对每种类型调用 `generate_dataset_parallel(raft_type, 250000)`
   - 打印总结: 总样本数、总失败率、4个HDF5文件路径
</action>
