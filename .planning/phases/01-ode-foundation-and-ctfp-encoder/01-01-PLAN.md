3. Create `src/raft_ode.py` implementing the full RAFT ODE system per D-01 through D-06:

   **State vector for single-equilibrium model (11 variables, per D-01):**
   `y = [M, I, CTA, P_dot, Int, mu0, mu1, mu2, lam0, lam1, lam2]`
   where mu = living chain moments, lam = dead chain moments.

   **Function `raft_ode_single_eq(t, y, kd, f, ki, kp, kt, kadd, kfrag)`:**
   - Species balances: dM/dt = -kp*P*M, dI/dt = -kd*I, dCTA/dt = -kadd*P*CTA + kfrag*Int, dP/dt = 2*f*kd*I + kfrag*Int - kadd*P*CTA - 2*kt*P^2, dInt/dt = kadd*P*CTA - kfrag*Int
   - Living moments: dmu0/dt = 2*f*kd*I/(kp*M+1e-30) ... (see Research Example 1 for exact equations; note: initiation creates chain of length 1, so contributes to mu0 += R_init, mu1 += R_init*1, mu2 += R_init*1)
   - Dead moments: dlam0/dt = kt*mu0*P + kadd*mu0*CTA (transfer creates dead chain), dlam1/dt = kt*mu1*P + kadd*mu1*CTA, dlam2/dt = kt*mu2*P + kadd*mu2*CTA
   - Per D-02: Mn = (mu1+lam1)/(mu0+lam0) * M_monomer, D = (mu2+lam2)*(mu0+lam0)/(mu1+lam1)^2

   **Function `raft_ode_preequilibrium(t, y, kd, f, ki, kp, kt, kadd, kfrag, kadd0, kfrag0)`:**
   - Extended state vector (13 variables): adds CTA_0 (initial RAFT agent) and Int_pre (pre-equilibrium intermediate) per D-03
   - Pre-equilibrium reactions: P. + CTA_0 -> Int_pre (kadd0), Int_pre -> R. + macro-CTA (kfrag0)
   - R. reinitiates as new propagating radical
   - Main equilibrium: P. + macro-CTA -> Int -> P'. + macro-CTA (kadd, kfrag) as in single-eq
   - The slow kfrag0 for dithioesters (kfrag0 << kfrag) creates the inhibition period

   **Function `simulate_raft(params, raft_type='ttc', t_end=36000, n_conv_points=50)`:**
   - `params` dict with keys: kd, f, ki, kp, kt, kadd, kfrag, M0, I0, CTA0, M_monomer (plus kadd0, kfrag0 for dithioester)
   - `raft_type` one of: 'dithioester', 'ttc', 'xanthate', 'dithiocarbamate'
   - Build initial conditions y0 from params (all moments start at 0, P_dot=0, Int=0)
   - For dithioester: use raft_ode_preequilibrium, set CTA_0=CTA0, CTA=0 initially (all RAFT agent is in initial form)
   - For others: use raft_ode_single_eq
   - Call solve_ivp with method='Radau', rtol=1e-8, per-component atol array (M:1e-6, I:1e-10, CTA:1e-8, P:1e-14, Int:1e-14, moments: 1e-14/1e-10/1e-6 for 0th/1st/2nd), dense_output=True, max_step=100.0
   - If not sol.success: return None
   - Sample at n_conv_points evenly-spaced conversions from 0.02 to 0.95 using scipy.optimize.brentq on dense_output to find time for each target conversion
   - Compute Mn and D at each conversion point per D-02
   - Normalize Mn: mn_norm = Mn / (M0/CTA0 * M_monomer) per Research normalization strategy (fixed denominator)

   **Function `compute_retardation_factor(params, raft_type, conv_target=0.5)`:**
   - Per D-06: Run simulate_raft WITH CTA and simulate a no-CTA FRP (set CTA0=0, kadd=0)
   - Find time when each system reaches min(conv_target, max_conv_raft)
   - Rp = d(conversion)/dt at that time, computed from dense_output as -dM/dt / M0
   - Return Rp_raft / Rp_frp (dimensionless, range 0-1)
   - If RAFT system reaches < 5% conversion, return 0.01 (near-zero sentinel per Research Pitfall 5)

   **Function `compute_inhibition_period(sol_dense, M0, t_end)`:**
   - Per D-05: Find time t_inh where conversion first reaches 1%
   - Return t_inh / t_end (dimensionless, range 0-1)
   - If conversion never reaches 1%, return 1.0
