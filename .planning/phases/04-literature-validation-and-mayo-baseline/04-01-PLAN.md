**1. Fixed parameter defaults** (per D-06, from dataset_generator.py center values):
```python
DEFAULT_FIXED_PARAMS = {
    'kp': 1000.0,        # geometric mean of 10^2 to 10^4
    'kt': 10**7.5,       # geometric mean of 10^6 to 10^9
    'kd': 1e-5,          # geometric mean of 10^-6 to 10^-4
    'f': 0.65,           # midpoint of 0.5-0.8
    'ki': 1e4,           # from TYPE_FIXED_PARAMS
    'I0': 0.01,          # midpoint of 0.001-0.05
    'M0': 1.0,           # from TYPE_FIXED_PARAMS
    'kfrag': 1e4,        # from dataset_generator.py
}

MONOMER_MW = {
    'Styrene': 104.15,
    'MMA': 100.12,
    'VAc': 86.09,
    'NVP': 111.14,
    'MA': 86.09,
    'BA': 128.17,
}
```

**2. `build_ode_params(row, ctr_value)`** — Construct ODE params dict from a literature CSV row:
- Takes a pandas Series (one CSV row) and a Ctr value
- Computes `kadd = ctr_value * DEFAULT_FIXED_PARAMS['kp']`
- Sets `CTA0 = cta_ratio * M0` where `cta_ratio = 0.01` (typical literature value)
- Sets `M_monomer` from `MONOMER_MW[row['monomer']]`
- For dithioester: adds `kadd0 = kadd * 0.1`, `kfrag0 = 1.0` (matching dataset_generator.py)
- Returns complete params dict ready for `simulate_raft()`

**3. `generate_simulated_data(row, sigma=0.03, seed=None)`** — Per D-07, generate simulated "experimental" data for one literature point:
- Calls `build_ode_params(row, row['ctr'])` to get true params
- Maps raft_type for ODE: if `row['raft_type'] == 'trithiocarbonate'` pass `raft_type='ttc'` to simulate_raft, else pass as-is
- Calls `simulate_raft(params, raft_type=mapped_type)`
- If result is None, return None
- Adds multiplicative Gaussian noise to Mn: `mn_noisy = mn * (1 + rng.normal(0, sigma, size=mn.shape))`
- Clips dispersity >= 1.0
- Returns dict with keys: `'conversion'`, `'mn'`, `'dispersity'`, `'mn_norm'`, `'cta_ratio'` (= CTA0/M0)

**4. `mayo_fit_ctr(target_conv, target_mn, row, bounds=(0.01, 20000))`** — Per D-05, single-parameter ODE fitting:
- Inner loss function `_loss(ctr_candidate)`:
  - Calls `build_ode_params(row, ctr_candidate)`
  - Maps raft_type for ODE (same as above)
  - Calls `simulate_raft(params, raft_type=mapped_type)`
  - If result is None, returns `1e10`
  - Interpolates simulated Mn to target conversion grid: `np.interp(target_conv, result['conversion'], result['mn'])`
  - Returns `np.mean((mn_interp - target_mn)**2)`
- Uses `minimize_scalar(_loss, bounds=bounds, method='bounded', options={'xatol': 1e-3})`
- Returns `result.x` (fitted Ctr value)

**5. `fold_error_log(ctr_pred, ctr_true)`** — Per D-08:
- `return 10 ** np.abs(np.log10(ctr_pred) - np.log10(ctr_true))`

**6. `fold_error_ratio(ctr_pred, ctr_true)`** — Per D-08:
- `return np.maximum(ctr_pred / ctr_true, ctr_true / ctr_pred)`

**7. `compute_summary_stats(ctr_true, ctr_pred)`** — Per D-09:
- Returns dict with keys: `'median_fold_error'`, `'pct_within_2x'`, `'pct_within_10x'`, `'rmse_log10'`, `'r2_log10'`
- Uses `fold_error_log` for fold-error metrics
- Uses `r2_score_np` and `rmse_np` from `utils.metrics` for log10 space metrics

**8. `ml_predict_single(model, sim_data, device='cpu', bootstrap_ckpt=None, cal_factors=None)`** — ML inference for one literature point:
- Constructs ctFP input: builds data tuples `(cta_ratio_norm, conv, mn_norm, disp)` where `cta_ratio_norm = sim_data['cta_ratio'] / 0.1` (matching training normalization)
- Calls `transform(data_tuples)` to get `(2, 64, 64)` tensor
- Adds batch dim: `fp_tensor = fp.unsqueeze(0)`
- If `bootstrap_ckpt` is not None and `cal_factors` is not None:
  - Calls `predict_with_uncertainty(model, fp_tensor, bootstrap_ckpt, cal_factors, device)`
  - Returns `(mean_pred, half_width)` where `mean_pred[0]` is log10(Ctr)
- Else:
  - `model.eval()`, `with torch.no_grad(): pred = model(fp_tensor.to(device)).cpu().numpy().squeeze()`
  - Returns `(pred, None)`

**9. `run_validation_pipeline(csv_path, model_path, bootstrap_path=None, calibration_path=None, output_dir='figures/validation', sigma=0.03, device='cpu', seed=42)`** — End-to-end orchestration:
- Loads CSV with `pd.read_csv(csv_path)`
- Loads model: `model = SimpViT(); model.load_state_dict(torch.load(model_path, map_location=device, weights_only=True)); model.eval()`
- Optionally loads bootstrap: `torch.load(bootstrap_path, map_location=device, weights_only=False)` and `json.load(calibration_path)`
- For each row in CSV (with tqdm progress):
  - `sim_data = generate_simulated_data(row, sigma=sigma, seed=seed+row['id'])`
  - If sim_data is None: log warning, skip
  - `ml_pred, ml_ci = ml_predict_single(model, sim_data, device, bootstrap_ckpt, cal_factors)`
  - `mayo_ctr = mayo_fit_ctr(sim_data['conversion'], sim_data['mn'], row)`
  - Store results in lists
- Build results DataFrame with columns: `id, raft_type, method, ctr_true, log10_ctr_true, ml_log10_ctr, mayo_log10_ctr, ml_fold_error, mayo_fold_error, ml_ci_low, ml_ci_high`
  - `ml_log10_ctr = ml_pred[0]` (first output is log10_Ctr)
  - `mayo_log10_ctr = np.log10(mayo_ctr)`
  - `ml_fold_error = fold_error_log(10**ml_pred[0], row['ctr'])`
  - `mayo_fold_error = fold_error_log(mayo_ctr, row['ctr'])`
  - CI columns: if ml_ci is not None, `ml_ci_low = ml_pred[0] - ml_ci[0]`, `ml_ci_high = ml_pred[0] + ml_ci[0]`; else NaN
- Compute summary stats for ML and Mayo separately via `compute_summary_stats()`
- Save results CSV to `{output_dir}/validation_results.csv`
- Save summary JSON to `{output_dir}/validation_summary.json`
- Call `plot_parity_ml_vs_mayo()` (see below)
- Return `(results_df, summary_dict)`
